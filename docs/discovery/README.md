## Relación con los documentos de solución

Los documentos de discovery (esta carpeta) definen **qué** construir y **por qué**.  
Los documentos de solución ([../solution/](../solution/)) definen **cómo** construirlo (decisiones técnicas y de diseño).

Ambas carpetas son complementarias y deben mantenerse alineadas.
